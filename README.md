# Проект: Сервис Mesto

## Описание проекта
Интерактивная страница об интересных местах России с возможностью добавления/удаления фотографий и простановкой лайков.

## Используемые технологии и методологии
* HTML
* CSS
* JS
* БЭМ
* Валидация на клиенте
* Webpack
* Babel
